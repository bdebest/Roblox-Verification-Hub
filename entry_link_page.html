Roblox Verification Hub (Backend Service)

This application is a dedicated **backend service** used for secure OAuth identity verification. It does not function as a general-purpose, public-facing website. Its only role is to process user authentication initiated by a third-party Discord bot.

If you were redirected here by a Discord bot, please return to the bot's channel to complete your verification process.
